PASO 1: CREAR COLECCIÓN "usuarios"
================================================================================

PARTE A: Crear la colección
──────────────────────────

1. Click en botón "+ Crear colección"
2. Escribir nombre: usuarios
3. Click "Siguiente"

PARTE B: Agregar primer documento (Encargado)
─────────────────────────────────────────────

Firestore te pide:
  ☐ "ID del documento"
  ☐ "Campos"

En "ID del documento":
  - Dejar en blanco o escribir: encargado1
  - Click "Guardar"

Aparecerá una ventana para agregar campos:

Click "Agregar campo":

CAMPO 1: nombre
  Nombre: nombre
  Tipo: String
  Valor: Juan Pérez
  Click ✓

CAMPO 2: email
  Nombre: email
  Tipo: String
  Valor: juan@taller.com
  Click ✓

CAMPO 3: activo
  Nombre: activo
  Tipo: Boolean
  Valor: true (marcar checkbox)
  Click ✓

CAMPO 4: rol
  Nombre: rol
  Tipo: String
  Valor: ADMIN
  Click ✓

CAMPO 5: fechaCreacion
  Nombre: fechaCreacion
  Tipo: Timestamp
  Valor: Click en reloj 🕐
    → Seleccionar: 2026-04-16
    → Seleccionar hora: 10:00 AM
    → Click "Actualizar"
  Click ✓

RESULTADO ESPERADO en Firestore:

┌──────────────────────────────────────────────┐
│ + Agregar campo                              │
└──────────────────────────────────────────────┘

1. Click en "+ Agregar campo"

APARECERÁ:
┌──────────────────────────────────────────────┐
│ Nombre del campo: [_____________]            │
│                                              │
│ Tipo de dato:    [Seleccionar ▼]            │
│                                              │
│ Valor:           [_____________]            │
│                                              │
│              [✓ Guardar]  [✗ Cancelar]      │
└──────────────────────────────────────────────┘

2. Completar:
   - Nombre del campo: (ej: "nombre")
   - Tipo de dato: (ej: "String")
   - Valor: (ej: "Juan Pérez")

3. Click "Guardar"


TIPOS DE DATO DISPONIBLES:

┌─────────────────┬──────────────────────────────────┐
│ TIPO            │ DESCRIPCIÓN                      │
├─────────────────┼──────────────────────────────────┤
│ String          │ Texto: "Juan", "Lija #120"      │
│ Number          │ Números: 50, 3, 2.5             │
│ Boolean         │ Verdadero/Falso: ☑ true/☐ false│
│ Timestamp       │ Fecha/Hora: Click reloj 🕐      │
│ Reference       │ Vínculo: usuarios/encargado1    │
│ Array           │ Lista: ["item1", "item2"]       │
│ Map             │ Objeto: {clave: valor}          │
│ Null            │ Vacío                           │
└─────────────────┴──────────────────────────────────┘


================================================================================
CÓMO TRABAJAR CON TIPOS ESPECIALES
================================================================================

TIMESTAMP (Fecha y Hora):
  1. Click en tipo "Timestamp"
  2. Se abre un selector de fecha/hora
  3. Selecciona: Día (ej: 16), Mes (ej: Abril), Año (ej: 2026)
  4. Selecciona hora: 09:00 AM
  5. Click "Actualizar"

ARRAY (Lista):
  1. Click en tipo "Array"
  2. Se abre ventana para agregar elementos
  3. Para vacío: dejar sin elementos y guardar
  4. Para con elementos:
     - Click "+ Agregar elemento"
     - Escribir valor: "E001"
     - Click ✓
     - Repetir para más elementos

REFERENCE (Referencia):
  1. Click en tipo "Reference"
  2. Escribir el path completo: colaboradores/col_001
  3. Guardar

BOOLEAN (Verdadero/Falso):
  1. Click en tipo "Boolean"
  2. Marcar ☑ para true
  3. Desmarcar ☐ para false
  4. Guardar

PRÓXIMOS PASOS DESPUÉS DE LLENAR FIRESTORE
================================================================================

1. ☐ Descargar credenciales JSON desde Firebase
   → Firebase Console → Configuración del proyecto → Cuentas de servicio
   → Botón "Generar clave privada" (JSON)
   → Guardar en tu proyecto como: serviceAccountKey.json

2. ☐ Crear archivo .env con las credenciales

3. ☐ Crear carpeta /src con estructura:
   /src
     /config
     /controllers
     /routes
     /middlewares
     /services

4. ☐ Empezar a codificar Express.js
